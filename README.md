# Koishi_p-MIDI
用来配布已发布的翻调作品的MIDI文件

## 关于文件前缀：
| 第一组方括号   | 第二组方括号 |     解释 |
| :-----         | :--:         | -------: |
|   N            |  N          | 无填词，没有分割\* |
|   Y            |  N          | 有填词，没有分割\* |
|   N            |  Y          | 无填词，有分割\* |
|   Y            |  Y          | 有填词，有分割\* |

\*指是否有为VOCALOID和UTAU等需要将连音分开填词的编辑器做适配，例如「せ｜ん」或「あ｜い」
